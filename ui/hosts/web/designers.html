AmbiTap designers — room~ and xtc~

room designer — drag S / L in plan or side; drag the far walls to resize

xtc designer — drag a speaker (angle = span, radius = distance); slider = regularization

The room pane draws the Allen–Berkley picture from the same math dsp::room uses (image cloud, reflectogram vs the RT60 slope). The XTC pane redesigns dsp::xtc in WASM on release and plots the shipped FIRs' responses plus the performance matrix P = C·H against the same KEMAR plant the design inverted — the ipsilateral-flatness and contralateral-rejection picture of the verification gates.
