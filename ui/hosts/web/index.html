AmbiTap soundfield — encode → rotate → analyze, live via WASM

Two test sources run through the verified C++ core compiled to WebAssembly inside an AudioWorklet: dsp::encoder (order 3) → SH rotation (the embedded profile's compute_sh_rotation + sh_block_applier) → analysis::soundfield_grid + energy_vector, plus an ALLRAD decode matrix for per-speaker metering.
